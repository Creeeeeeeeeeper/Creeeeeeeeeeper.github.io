Docker学习

安装

Docker 下载

这里使用树莓派

把第一条和第四条执行以下，下载不了的话就挂梯子

命令

docker pull

此命令用来从仓库下载镜像

docker images

此命令用户列出所有下载过的docker镜像

docker rmi

此命令用户删除一个docker镜像，docker remove image

docker run

此命令用于创建并运行一个容器

detached mode，可以不阻塞当前窗口，在后台运行

运行docker run 命令时，如果镜像不存在，那么会自动拉取

-p 80:80

-v 挂载目录

在我的宿主机上有一个apache服务器，docker里面又启动了一个nginx服务器，我要nginx也是用apache的服务器内容，就可以这样挂载，宿主机和容器对文件的修改都会影响宿主机原本的文件内容。

docker ps

查看已下载的镜像

docker inspect

查看容器信息

docker exec

使用docker exec -it NAME /bin/sh可以进入一个容器，在容器里面执行linux命令

由于docker为了尽量压缩容器大小，所以很多工具都是没有的，先cat /etc/os-release看一下这个docker容器是什么发行版的

看到是Debian，所以下载工具是apt，先apt update更新一下索引
